```

```

```bash
docker-compose up
```

## Описание
Поднимается БД -> далее отрабатывает скрипт который .
- скачивает данные(источник данных заменил так как по тз можно был).
- Разархивирует архив.
- Сохраняет данные в БД.
- Логирование происходит в  [файл](log_file.log).
